ABOUT ME

שמי קרסיק דז'נה, נולדתי בשנת 1937, בעיר בירובידז'אן שהיא אוטונמיה יהודית ברוסיה

איפה גרתי

אבי היה איש צבא וכמו כן הייתי נשואה לאיש צבא, לכן עברתי הרבה מדינות במהלך השנים
לאחר הולדתי ברוסיה, עברנו לעיר פאנבז'יס שבלטביה. בזמן המלחמה, כיוון שאבא שלי היה קצין בצבא האדום, פינו אותנו למחוז פנזסק ברוסיה, שם עוד לא הגיעו הנאצים. כיוון שרוב משפחתה של אימי גרה במינסק (בלארוס), חזרנו לשם בשנת 1944, השנה בה בלארוס שוחררה מהנאצים. בשנים 1944-1946 גרנו ברומניה בשל שליחות צבאית של אבי. בשנת 1946 העבירו את אבי לשרת ברוסיה ולאחר מכן בשנת 1955 עברנו לבלארוס, לעיר מינסק. בשנת 1958 התחתנתי עם איש צבא, ועברנו לעיר באקו באזרבז'אן. שם גרנו עד שנת 1980. לאחר מכן חזרנו למינסק שבבלארוס. בשנת 1995, לאחר ששני הילדים שלי עלו לארץ, עלינו בעקבותם וגרנו בבת ים.
היום אני אמא גאה ל-2 בנים וסבתא ל-3 נכדים, מתגוררת בבית גיל הזהב באשקלון

1. The Three main diffrences between web1,web2 and web3

in web 1.0 the content is given by website's managers, in web 2.0 sharing and communication is available, in web 3.0 the information can be read by human and computer (using AI)

web 1.0 using links and directories in search engine, in web 2.0 tagging is used, in web 3.0 the focus on learning user behavior

web 0.1 focus on companies, web 2.0 focus on community and web 3.0 focus on individualuty
